🎯 Step 1：開 BP_TrackSpline 加路軌

1. 喺 Content Browser（左下角），㩒路徑：
Blueprints → Core
2. 雙擊 BP_TrackSpline 打開佢
（會彈出 Blueprint Editor 新視窗）
3. 㩒「Add Component」按鈕（左上角綠色 + 號）
4. 喺 Search bar 打 Spline
㩒 Spline Component
5. 喺右邊 Details Panel，將 Name 改做：
TrackSpline
6. 㩒工具列 「Compile」（藍色按鈕）→ 然後 「Save」
✅ 完成！可以 Close 視窗

🎯 Step 2：開 BP_MineCart 加礦車 Component

1. 返 Content Browser → Blueprints → Core
雙擊 BP_MineCart

2a. 加 CartMesh（礦車車身）

㩒 Add Component → Search Static Mesh
㩒 Static Mesh Component → 改名 CartMesh
Location 設做：X=0, Y=0, Z=50

2b. 加 CameraArm（鏡頭手臂）

㩒 Add Component → Search Spring Arm
㩒 Spring Arm Component → 改名 CameraArm
Location 設做：X=0, Y=-300, Z=200
Rotation 設做：Pitch=0, Yaw=-10, Roll=0
喺右邊 Details Panel，向下 scroll 搵 Lag 欄位：
☑ Enable Camera Rotation Lag
☑ Enable Camera Lag
Camera Lag Speed = 5.0

2c. 加 PlayerCamera（玩家鏡頭）

㩒 Add Component → Search Camera
㩒 Camera Component → 改名 PlayerCamera
重要：拖住 PlayerCamera，放入 CameraArm 下面（變成佢嘅 child）
Location 全部 = 0
最後㩒 Compile → Save
✅ BP_MineCart 完成！Close 視窗

🎯 Step 3：開 BP_NarratorStation 加車站 Component

1. Content Browser → Blueprints → Stations
雙擊 BP_NarratorStation

3a. 加 TriggerZone（感應區）

㩒 Add Component → Search Box
㩒 Box Collision → 改名 TriggerZone
Location：X=0, Y=0, Z=100
Box Extent：X=200, Y=200, Z=200
Collision Enabled：揀 Query Only (No Physics Collision)

3b. 加 NarrationAudio（語音播放）

㩒 Add Component → Search Audio
㩒 Audio Component → 改名 NarrationAudio
Auto Activate：❌ 唔好 tick（我哋用程式控制播放）

3c. 加 StationLabel（站名標籤）

㩒 Add Component → Search Text Render
㩒 Text Render Component → 改名 StationLabel
Location：X=0, Y=0, Z=400
Text：打 車站（之後程式會改）
World Size：80
Color：㩒色揀 淺金色（R=255, G=255, B=200）
Horizontal Alignment：Center
最後㩒 Compile → Save
✅ BP_NarratorStation 完成！Close 視窗

✅ 完成 Check List
